## 使用方法

> 1. 前端前后台 npm i 完成之后 使用 npm run dev 
> 2. 后端，当时没考虑清除，导致后端代码有点乱... 具体使用 npm i 之后，分别启动相应的服务 `nest start -w admin`(前端后台) / `nest start -w server`(前端前台)
> 3. 上面两项运行完毕之后，导入 admin.json 到 Mongodb 数据库
